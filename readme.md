
Modified version of https://github.com/joonatan-r/nethack for replaying the playback files from it, available at https://joonatan-r.github.io/nethack-player/ or https://joonatan-r.github.io/nethack-player/#pc.

Usage: upload the file(s) generated by the game (in order), then click play. Space can be used for pause/unpause, number input can be given for skipping a certain number of events, and escape resets the number. Arrow right or "l" can be used to execute the skip forward, and arrow left or "h" can be used to execute skip backwards (Note: slower than forward with current implementation).
