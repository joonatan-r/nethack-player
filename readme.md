
Modified version of https://github.com/joonatan-r/nethack for replaying the playback files from it.

Usage: upload the file(s) generated by the game, then click play. Space can be used for pause/unpause, and number input can be given for skipping a certain number of events. Arrow right or "l" can be used to execute the skip forward, and arrow left or "h" can be used to execute skip backwards (Note: slower than forward with current implementation).
